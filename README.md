# Rodar a aplicação
Após configurar o arquivo .env, você pode rodar a aplicação com o comando:

npm start
Isso iniciará o servidor Node.js e conectará ao banco de dados MySQL usando as credenciais fornecidas.

Para rodar os testes
npm test
